<h1 align="center">NodeJS Simple Logger</h1>

> A simple logger for node.js

### [Github](https://github.com/JamieGrimwood/nodejs-simple-logger)

## Install

```sh
npm install nodejs-simple-logger
```

## Author

Example code:

```javascript
const log = require("nodejs-simple-logger")

log.Info('test')
log.Warn('test')
log.Error('test')
log.Success('test')
```
